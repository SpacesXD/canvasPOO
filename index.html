API
Métodos que canvasPOO agrega al ctx.
ctx.circle(x, y, r)forma

Círculo relleno. Usa el fillStyle actual.

paramtipodesc
xnumberCentro X
ynumberCentro Y
rnumberRadio
ctx.fillStyle = 'red'
ctx.circle(200, 200, 40)
ctx.strokeCircle(x, y, r)forma

Borde de círculo. Usa strokeStyle y lineWidth actuales.

ctx.strokeStyle = 'cyan'
ctx.lineWidth = 2
ctx.strokeCircle(100, 100, 30)
ctx.roundRect(x, y, w, h, r?)forma

Rectángulo con esquinas redondeadas. r default: 8.

ctx.fillStyle = '#4db8ff'
ctx.roundRect(50, 50, 120, 60, 12)
ctx.line(x1, y1, x2, y2)forma

Línea directa entre dos puntos.

ctx.strokeStyle = 'white'
ctx.line(0, 0, 400, 300)
ctx.polygon(points)forma

Polígono cerrado desde array de [x,y].

ctx.fillStyle = 'lime'
ctx.polygon([[200,50],[280,200],[120,200]])
ctx.background(color)utilidad

Rellena todo el canvas. Úsalo al inicio de cada frame.

ctx.background('#111')
ctx.clear()utilidad

Borra todo el canvas dejándolo transparente.

ctx.clear()
ctx.print(text, x, y, opts?)utilidad

Texto con todas las opciones en un llamado. Restaura el ctx después.

opcióndefaultdesc
size16Tamaño px
family'sans-serif'Fuente
color'#fff'Color
align'left'left/center/right
baseline'top'top/middle/bottom
boldfalseNegrita
ctx.print('hola', 20, 20, { size:24, color:'#4db8ff', bold:true })
ctx.withAlpha(a, fn)utilidad

Ejecuta fn con globalAlpha temporal. Ahorra save/restore manual.

ctx.withAlpha(0.3, () => {
  ctx.circle(200, 200, 60)
})
ctx.linearGradient(x1,y1,x2,y2, stops)utilidad

Gradiente lineal listo para asignar a fillStyle. stops: [[pos, color], ...]

ctx.fillStyle = ctx.linearGradient(
  0,0, 0,400,
  [[0,'#4db8ff'],[1,'#ff6b9d']]
)
ctx.radialGradient(x,y,r1,r2, stops)utilidad

Gradiente radial desde r1 hasta r2 centrado en (x, y).

ctx.fillStyle = ctx.radialGradient(
  200,200, 0,100,
  [[0,'white'],[1,'transparent']]
)
ctx.Rect(x, y, w, h, color?)objeto

Rectángulo como objeto. Pivote en el centro para rotate/scale.

position.x/ysize.w/hscale.x/yrotationalphavisiblecolordraw()
const p = ctx.Rect(100,100,40,40,'red')
p.position.x += 5
p.rotation = 0.3
p.scale.x = 2
p.draw()
ctx.Circle(x, y, r, color?)objeto

Círculo como objeto.

position.x/yradiusscale.x/yalphavisiblecolordraw()
const b = ctx.Circle(200,200,20,'cyan')
b.position.y += 3
b.radius = 30
b.draw()
ctx.Polygon(points, color?)objeto

Polígono como objeto con posición, rotación y escala.

position.x/ypointsscale.x/yrotationalphavisibledraw()
const t = ctx.Polygon([[0,-20],[20,20],[-20,20]],'lime')
t.position.x = 300
t.rotation += 0.01
t.draw()
ctx.Sprite(x, y, w, h, img)objeto

Imagen como objeto. img debe ser un HTMLImageElement ya cargado.

position.x/ysize.w/hscale.x/yrotationalphavisibleimgdraw()
const img = new Image()
img.src = 'hero.png'
img.onload = () => {
  const s = ctx.Sprite(100,100,64,64, img)
  s.draw()
}
ctx.Text(x, y, text, opts?)objeto

Texto como objeto. Cambia .text en el loop sin recrear nada.

position.x/ytextsizecolorboldalignalphavisibledraw()
const sc = ctx.Text(20,20,'0',{ size:24, bold:true })
sc.text = `${puntos}`
sc.draw()

01

Setup básico

Canvas, loop y tu primer objeto

PASO 1

Carga canvasPOO

Pon el script antes de tu main.js.

<script src="https://unpkg.com/canvas-poo@1.0.2/canvasPOO.js"></script>
<script src="main.js"></script>
PASO 2

Envuelve el ctx

const ctx = canvasPOO(canvas.getContext('2d'))
PASO 3

Crea objetos y dibuja en loop

const caja = ctx.Rect(100,100,60,60,'#4db8ff')
let last = 0
function loop(now) {
  const dt = (now-last)/1000; last = now
  caja.rotation += dt
  ctx.background('#050810')
  caja.draw()
  requestAnimationFrame(loop)
}
requestAnimationFrame(loop)

PASO 4

Swipe táctil

let t0 = null
canvas.addEventListener('touchstart', e =>
  t0 = {x:e.touches[0].clientX, y:e.touches[0].clientY})
canvas.addEventListener('touchend', e => {
  const dx=e.changedTouches[0].clientX-t0.x
  const dy=e.changedTouches[0].clientY-t0.y
  dir = Math.abs(dx)>Math.abs(dy)
    ? (dx>0?{x:1,y:0}:{x:-1,y:0})
    : (dy>0?{x:0,y:1}:{x:0,y:-1})
})
03

Partículas

Explosiones con Circle, alpha y escala

PASO 1

Crear una partícula

function spawn(x, y) {
  const p = ctx.Circle(x, y, Math.random()*6+2, '#4db8ff')
  p.vx = (Math.random()-.5)*200
  p.vy = (Math.random()-.5)*200
  p.vida = 1
  return p
}
PASO 2

Update y draw en el loop

parts = parts.filter(p => p.vida > 0)
parts.forEach(p => {
  p.vida -= dt
  p.position.x += p.vx * dt
  p.position.y += p.vy * dt
  p.alpha = p.vida
  p.draw()
})
PASO 3

Emitir al click

canvas.addEventListener('click', e => {
  const r = canvas.getBoundingClientRect()
  for (let i=0;i<20;i++)
    parts.push(spawn(e.clientX-r.left, e.clientY-r.top))
})
